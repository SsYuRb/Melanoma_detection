# Melanoma_detection


Repository contain code for boundary prediction in class `ExcisionPredictor`. 
To check it prepare environment:
    python -m venv venv
    venv/bin/python -m ensurepip
    venv/bin/pip install -r requirements.txt
    
And run unittest:

  python -m unittest
